# TCS NQT Verbal Practice Simulator

A high-fidelity React.js simulation engine replicating the strict structural formatting rules and workflows of the latest TCS NQT Verbal Ability assessment platform.

## Architectural Components

* **Email Writing Module**: 10-minute timer interface checking syntax strings against 20 structured scenario banks. Evaluation measures subject presence, opening/closing protocols, word limits, and specific keyword densities.
* **Passage Recall Module**: Exactly 30 seconds reading gate logic that dynamically wipes the target content array before loading memory input frames. Evaluates on precise sub-string correlation matrices across 30 domain datasets.
* **Unified Mock Mode**: Simulates real-world conditions by executing testing gates end-to-end sequentially.
* **Local Engine Diagnostics**: Native storage persistence mapping analytical telemetry data directly inside standard local dashboard widgets and community leaderboards.
